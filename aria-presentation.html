Zero Friction Setup

Get Started in Minutes

Aria works with your existing AI IDE. No complex installation, no dependencies.

🌐

Automatic Chrome MCP Setup

Browser automation configured automatically during first QA run

Aria automatically detects and configures Chrome MCP for your AI platform. No manual installation required—just enable QA in your environment.

# Aria-Agent/.env - Enable automated browser QA QA_ENABLE_CHROME_MCP="true" QA_DEV_SERVER_URL="http://localhost:5173" QA_TEST_EMAIL="test@example.com" QA_TEST_PASSWORD="password123"

Step 1: Configure Environment

Set up your Aria-Agent/.env file with Azure credentials and QA settings.

# Azure DevOps Configuration AZURE_ORG="your-org" AZURE_PROJECT="your-project" AZURE_PAT="your-pat-token" AZURE_ASSIGNEE="Your Name"

Step 2: Wake Up Aria

Use any of these wake phrases in your AI IDE to activate Aria.

"Hey Aria, good morning" "Hey Aria, wake up" "Aria, triage my tickets" "Aria, fix ticket #123456"

Aria reads Aria-Agent/ARIA-AUTONOMOUS.md and executes the entire pipeline autonomously.
